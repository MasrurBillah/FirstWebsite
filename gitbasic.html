• Create a file: echo "# Data Science" >> README.md
  • Start Git: git init
  • Adding a file in a repository: git add README.md
  • Adding everything: git add .
  • Doing Commit (must do before uploading): git commit -m "first commit"
  • Repository link: git remote add origin https://github.coml/ads.git
  • Upload in GitHub: git push -u origin master
  • Create a branch: git checkout -b [branch_name]
  • Upload or download from github: git push ...and... git pull
